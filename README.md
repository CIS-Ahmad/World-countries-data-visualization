# 🌍 World Countries Data Visualization

A dynamic data visualization application built with **HTML, CSS, and Vanilla JavaScript**.  
The application displays information about countries and provides visual comparisons of the world's most populated countries and most spoken languages using interactive charts and progress bars.

## 🌐 Live Demo

👉 Website: https://cis-ahmad.github.io/World-countries-data-visualization/

## 📖 About

The **World Countries Data Visualization** project is a JavaScript-based application that processes country data and transforms it into meaningful visual information.

Users can explore:
- The world's most populated countries.
- The most spoken languages worldwide.

The project uses JavaScript array methods, data processing techniques, and dynamic DOM manipulation to generate visual reports directly in the browser without external libraries.

This project was developed to practice:
- Data transformation.
- Array manipulation.
- Sorting algorithms.
- Dynamic HTML generation.
- JavaScript modules.
- Data visualization concepts.

## ✨ Features

- Display total number of available countries.
- Show the top 10 most populated countries.
- Calculate and display total world population.
- Show the top 10 most spoken languages.
- Dynamic progress bar visualization.
- Automatic sorting by population and language count.
- Generate HTML elements dynamically using JavaScript.
- Responsive and clean user interface.
- Interactive buttons to switch between different data views.
- Uses external JavaScript modules for data management.
